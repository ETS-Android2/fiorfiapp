# Fiorfi  Mobile App

A social media application with simple structures written in Java language.


## Features

- Login, Signup, Forgot password (with OTP verification)
- Follow, Unfollow, Profile views.
- Post photo posts from camera or gallery
- Edit your profile (username, Bio, Name).
- Get Notifications.
- Search for users
- Like posts
- Comment on posts
- View all comments on a post
- View all followers, following
